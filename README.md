# Safe Calculator Windows

Safe Calculator adalah sebuah alat yang digunakan untuk membantu pelajar menghitung lebih cepat saat berhadapan dengan soal Matematika.

## Tujuan
Sebagai alat bantu bagi murid yang memiliki masalah menghitung agak lambat, padahal sangat paham rumus.

## Fitur
- Operator +,-,*,/ yang memudahkan murid dalam menghitung, namun tetap memastikan bahwa murid tetap paham rumus.

## Cara Menjalankan
1. Copy code ke notepad.
2. Save sebagai Save Calculator.bat.
3. Lalu tinggal klik file nya dan code akan jalan.

## Contoh
Jika seorang murid berhadapan dengan soal Matematika, dan soal itu memerlukan rumus, alih - alih menghitung tambah, kurang, kali, bagi secara manual, 
sebaliknya gunakan Safe Calculator
